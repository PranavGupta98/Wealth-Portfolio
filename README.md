# WealthTracker (Wealth-Portfolio)

> *Vibe Coding a Wealth Tracker as a 9-to-5 income investor*

## 🎯 Purpose & Motivation

**WealthTracker** is a personal, offline-first wealth management application. The motivation behind this project stems from a desire for a simple, private, and secure way to track investments and monitor financial growth without relying on bloated cloud services or handing over sensitive financial data to third-party applications.

Built around the **KISS** (Keep It Simple, Stupid) principle, this project guarantees:
- **Data Privacy:** 100% local data persistence. Your financial data never leaves your machine.
- **Simplicity:** A clean and modern frontend designed to focus purely on the numbers that matter.
- **Independence:** Avoiding heavy cloud dependencies and vulnerable third-party APIs.

## 🚀 Features

- **Local Express Backend:** A lightweight local server that safely handles data and requests.
- **Live Price Updates:** Integration with Yahoo Finance to pull real-time investment pricing directly to your dashboard.
- **Modern UI:** Built with Vite and React for a fast and beautiful user experience.

## 🚧 Status: Under Development

This project is currently **under active development**. 

Expect to see frequent updates as new features, components, and enhancements are regularly pushed to the repository. Upcoming features include enhanced portfolio analytics, improved historical charting, and deeper categorization for different asset classes.
